# Projeto: Migração de Banco de Dados Oracle 11g para 19c

**Papel:** Scrum Master  
**Período:** 2024  
**Contexto:** Projeto estratégico de modernização da infraestrutura de dados corporativos.  

### Desafio
Garantir a migração sem downtime significativo e com integridade total dos dados.  

### Solução
Planejamento ágil integrado entre times de banco de dados e desenvolvimento; execução controlada por sprints técnicas e validações incrementais.  

### Resultados
- Migração bem-sucedida de 20 bases de dados críticas  
- Zero perda de dados e melhoria de performance de 40%
